🧠 Headache Diary / Дневник головной боли
Веб-приложение для трекинга приступов головной боли с визуализацией данных.
Портфолио-проект врача-невролога, осваивающего фронтенд.


🎯 Цель проекта (техническая)
Практическое освоение Vue 3 (Composition API)

Построение полнофункционального SPA (Single Page Application)

Демонстрация навыков работы с состоянием, маршрутизацией, UI/UX

Главное: Создание работающего прототипа для HealthTech-портфолио


📊 Предметная область (ваше врачебное преимущество)
Приложение учитывает клинически значимые параметры:

Интенсивность боли (по ВАШ, визуальная аналоговая шкала)

Локализация (гемикрания, лобная, затылочная и др.)

Характер (пульсирующая, давящая, жгучая)

Сопутствующие симптомы (фотофобия, тошнота, аура)

Триггеры (стресс, погода, пища, сон)

Принятые лекарства с оценкой эффективности

Сезонность и циркадные ритмы (время суток)


🗺️ Дорожная карта изучения Vue через проект
Фаза 1: Основа (Vue 3 + Composition API)
Задача: создать статический прототип

Инициализация проекта c использованием сборщика Vite (npm create vite@latest)

Базовая структура компонентов: App.vue

Реактивность: ref() 

Обработка событий: @click кнопок

Директивы:v-if для условного отображения
